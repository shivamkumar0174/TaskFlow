# TaskFlow — Smart Task Management Application

TaskFlow is a modern and interactive task management web application designed to help users organize, track, and manage their daily activities efficiently. Developed using HTML, CSS, and JavaScript, the application focuses on delivering a clean user experience with real-time task updates and responsive design principles.

The application allows users to create, edit, delete, and mark tasks as completed seamlessly. It also features a dynamic progress tracking system that visually represents task completion status, improving productivity and workflow management.

To provide a more realistic and professional user experience, TaskFlow integrates the Local Storage API, enabling tasks to remain permanently saved in the browser even after page refresh or reopening the application.

The project emphasizes modern frontend development practices such as DOM manipulation, event-driven programming, reusable functions, responsive UI design, and browser-based data persistence.

---

## 🚀 Core Features

- Create and manage tasks dynamically
- Edit existing tasks in real time
- Delete tasks instantly
- Mark tasks as completed with visual indicators
- Interactive progress bar for completion tracking
- Persistent data storage using Local Storage
- Responsive and modern user interface
- Keyboard interaction support using the Enter key
- Smooth user interactions and task updates
